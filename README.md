# 🛠️ Dev Tracker

**Track, Prioritize, Ship — Your Way.**  
A minimalist issue tracker for solo developers and small teams.

## 🚀 Features
- User registration and login
- Clean, responsive UI
- Create, edit, and delete tasks
- Status updates (e.g. "Done")
- Built with Flask + Next.js
